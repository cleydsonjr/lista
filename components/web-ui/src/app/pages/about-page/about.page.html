Sobre o aplicativo

Listas simples para WhatsApp

Já sofreu para montar uma lista de confirmados para o churrasco no grupo do WhatsApp?

Esse aplicativo é a solução definitiva para esse problema. Uma alternativa simples, gratuita e sem propagandas para compartilhamento de listas interativas. Só criar sua lista e compartilhar o link no grupo.

Por ser gratuito, existem algumas restrições: A quantidade de itens nas listas é no máximo 50 e as listas antigas (mais de 3 meses sem alterações) serão excluídas. Se você precisa que a lista fique disponível por mais tempo, provavelmente está usando a ferramenta errada, melhor exportar e salvar em um arquivo.
